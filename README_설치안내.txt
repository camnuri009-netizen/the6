온빌드 Firebase CMS 설치 안내

1. Firebase 프로젝트 생성
- https://console.firebase.google.com 접속
- 프로젝트 만들기
- Web App 추가
- firebaseConfig 코드 복사

2. firebase-config.js 수정
- 파일 안의 firebaseConfig 부분을 Firebase 콘솔에서 복사한 값으로 교체

3. Firestore Database 생성
- Firebase 콘솔 > Firestore Database
- 데이터베이스 만들기
- 테스트 모드로 시작

4. GitHub에 6개 파일 업로드
- index.html
- admin.html
- style.css
- site.js
- admin.js
- firebase-config.js

5. 관리자 접속
- https://onbuild.kr/admin.html
- 비밀번호 1234

이 버전은 Firebase에 저장되므로 PC, 내 휴대폰, 타인 휴대폰 모두 같은 블로그/카카오 링크와 이미지를 봅니다.
